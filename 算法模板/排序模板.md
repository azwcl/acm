## 快速排序算法模板
```cpp
void quickSort(int a[], int l, int r)
{
    if (l >= r) return;

    int i = l - 1, j = r + 1, x = a[(l + r ) >> 1];

    while(i < j)
    {
        do i++; while(a[i] < x);
        do j--; while(a[j] > x);
        if(i < j) swap(a[i], a[j]);
    }

    quickSort(a, l, j), quickSort(a, j + 1, r);
}
```
